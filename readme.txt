Easily embed Bookingmood widgets into your Wordpress website

== Description ==

Bookingmood allows you to manage bookings for your rental properties.
This plugin makes it easy for to embed Bookingmood widgets on your Wordpress website.

== Frequently Asked Questions ==

= How do I use this plugin? =

Using this plugin is very simple.

1. Sign up for an account on [Bookingmood](https://bookingmood.com).
2. Follow the steps there to create a widget.
3. Once you have created a widget, navigate to the "embed" tab and copy the oEmbed URL.
4. Now you can simply paste the link in the editor on the page or post you like.
5. You should see the widget appear!

= The widget does not seem to resize properly =

In the Wordpress page editor, widgets might not resize, causing scrollbars to be visible.
When you view the actual page however, resizing should work fine.
